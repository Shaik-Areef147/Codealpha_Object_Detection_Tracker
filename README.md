🚀 Object Detection and Tracking System

This project is a real-time Object Detection and Tracking web application built using YOLOv8, OpenCV, and Flask. Users can upload videos, detect multiple objects, track them across frames, and visualize results with bounding boxes and unique tracking IDs. The application features an attractive web interface and is deployed using Render.

🔹 Features
• Object Detection using YOLOv8
• Multi-Object Tracking
• Bounding Boxes and Labels
• Video Upload Support
• Flask-based Web Application
• Responsive and Colorful UI
• Render Deployment Ready
